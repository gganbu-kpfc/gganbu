# 깐부파트너스 홈페이지 프로젝트

깐부파트너스 공식 홈페이지 제작 프로젝트입니다.

이 홈페이지는 단순 회사 소개용 페이지가 아니라,
방문자가 첫 화면에서 회사 정체성을 이해하고,
직접 운영 브랜드와 서비스, 운영진, 신뢰 요소, 수익 구조를 확인한 뒤
문의로 자연스럽게 이어지도록 설계된 구조입니다.

---

## 1. 프로젝트 목적

이 홈페이지의 목적은 아래와 같습니다.

- 깐부파트너스가 어떤 회사인지 한눈에 이해시키기
- 직접 운영 브랜드 4개를 명확히 보여주기
- 운영진, 지역사회 활동, 방송 출연 등을 통해 신뢰를 만들기
- 브랜드별 서비스와 장점을 쉽게 설명하기
- 수익 구조와 FAQ를 통해 반감을 줄이기
- 방문자가 문의로 자연스럽게 이어지게 만들기

---

## 2. 브랜드 운영 구조

### 직접 운영 브랜드
깐부파트너스 산하에서 직접 운영하는 핵심 브랜드는 아래 4개입니다.

- 깐부마케팅
- 깐부AI솔루션
- KPFC
- PICKCAR

### MOU 협력사 / 협력 영역
아래 브랜드 및 영역은 직접 운영 브랜드가 아니라,
협력 구조로 연결되는 MOU 영역입니다.

- 깐부다이렉트
- 깐부폰
- 깐부렌탈
- 한국태양광연금협회
- 한국부업인협회

운영 방향은 다음과 같습니다.

> 모든 걸 직접 하려는 회사가 아니라,
> 잘하는 4개는 직접 운영하고,
> 나머지는 검증된 협력사와 연결하는 구조로 간다.

---

## 3. 사이트 운영 방향

### 핵심 방향
- 뜬구름 같은 문구 금지
- 실제 하는 일 중심
- 짧고 이해 쉬운 문장 사용
- 텍스트만 길게 나열하지 않고
  박스 / 이미지 / 가로 배너 / 인포그래픽을 섞어 구성
- 모바일에서도 바로 이해되는 구조

### 디자인 방향
- 화이트 중심의 신뢰형 디자인
- 전체 비율은 화이트 3 : 컬러 2 정도로 유지
- 메인 포인트 컬러는 네이비 / 블루 / 약한 보라톤
- 과한 장식보다 정리된 느낌 우선
- 돈, 컨설팅, 자동차, AI를 다루는 회사답게 무게감 있게 표현
